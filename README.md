# CashwaysPass 🚀

Bem-vindo ao repositório do **CashwaysPass** (Pass 2.0).

> Sistema robusto projetado com tecnologia de ponta para acompanhamento e gerenciamento analítico do fluxo financeiro educacional.

🔗 **[VER APRESENTAÇÃO COMPLETA DO PROJETO AQUI](APRESENTACAO_PROJETO.md)**

---

## 💻 Tech Stack
- Frontend: Angular + Tailwind CSS
- Backend: NestJS
- Database: Supabase PostgreSQL
- Autenticação: Supabase Auth

## 🔧 Run Locally

Para rodar este monorepo localmente, inicie os dois diretórios em rotinas assíncronas:

**Backend:**
```bash
cd backend-nest
npm install
npm run start:dev
```

**Frontend:**
```bash
cd frontend
npm install
npm start
```
